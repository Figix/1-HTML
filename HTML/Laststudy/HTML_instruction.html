크키, 색을 다루는 style 명령어

스타일 설명 사용법 예시
color 글자색상 지정 style="color:red"
background-color 배경색상 지정 style="background-color:red"
border 테두리 지정 style="border:3px solid #0a0a0a"
border-radius 테두리의 둥근 정도를 설정 style="border:3px solid #0a0a0a; border-radius: 12px;"
width,height 각각 가로,세로 길이 style="width:100px; height:50px; background-color:red;"
margin 외부 여백 지정 style="margin : 30px; background-color:red;"
padding 내부 여백 지정 style="padding:40px;background-color:red;"
